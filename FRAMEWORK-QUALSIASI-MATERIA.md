### Passo 4: Assegna i Punteggi

Usa una scala consistente (es. 1-9 o 1-10):

#### Scala Consigliata (1-9):
```
1-2: Assente o solo accenni superficiali
3-4: Concetti base o introduttivi elementari
5-6: Trattazione media, sufficientemente approfondita
7-8: Importante/essenziale, ben approfondito
9: Fondamentale, trattazione completa e avanzata
```

#### Con Note Descrittive (Consigliato):
```csv
Argomento,L-XX,L-YY,L-ZZ
Argomento 1,9 - Fondamentale,8 - Essenziale,6 - Medio
Argomento 2,7 - Importante,9 - Fondamentale,5 - Base
```

## üí° Consigli per Framework Efficaci

### ‚úÖ DO (Fai):

1. **Sii Specifico**
   ```csv
   ‚úÖ Buono: "Equazioni differenziali ordinarie del primo ordine"
   ‚ùå Troppo generico: "Equazioni"
   ```

2. **Usa Note Descrittive**
   ```csv
   ‚úÖ Buono: "9 - Fondamentale per corso base"
   ‚ùå Solo numero: "9"
   ```

3. **Mantieni Consistenza**
   ```csv
   ‚úÖ Buono: Sempre scala 1-9, sempre stessa struttura
   ‚ùå Incoerente: A volte 1-10, a volte solo testo
   ```

4. **Includi Collegamenti**
   ```csv
   Argomento,Punteggio,Prerequisiti,Note
   Derivate,9,Limiti,"Fondamentale per integrali"
   ```

---

### ‚ùå DON'T (Non Fare):

1. **Non Usare Righe Vuote All'Inizio**
   ```csv
   ‚ùå Errato:
   
   Argomento,Classe1,Classe2
   ```

2. **Non Mescolare Formati**
   ```csv
   ‚ùå Errato:
   Argomento,L-31,L-32
   Arg1,9 - Fondamentale,Otto
   Arg2,7,7 - Importante
   ```

3. **Non Dimenticare l'Encoding UTF-8**
   - Necessario per caratteri accentati italiani
   - In Excel: scegli "CSV UTF-8"

## üîß Risoluzione Problemi

### Problema: "CSV non valido"
**Soluzione:**
- Verifica che la prima riga contenga intestazioni
- Rimuovi righe vuote all'inizio
- Salva come UTF-8

### Problema: "Caratteri strani (ÔøΩ, ?, etc.)"
**Soluzione:**
- Salva il CSV con encoding UTF-8
- In Excel: "CSV UTF-8 (delimitato da virgole)"
- In Notepad++: Encoding ‚Üí UTF-8

### Problema: "Colonne non separate"
**Soluzione:**
- Usa la virgola come separatore
- Racchiudi testi con virgole tra virgolette: `"L-29, LM-13 Farmacia"`
